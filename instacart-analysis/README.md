## 📌 프로젝트 개요
Walmart, Uber Eats, DoorDash 등 경쟁 플랫폼의 증가로 고객 이탈 위험이 높아지는 상황.
단순 마케팅이 아닌, **고객 행동 데이터 기반으로 리텐션 분기점을 규명**하고
장기 고객으로 전환시키는 행동 구조를 설계하는 것을 목표로 했다.

## 🔬 분석 가설
> 사용자의 장기 유지 여부는 **'무엇을 구매했는가'** 보다
> **'얼마나 빠르게 반복 주문 리듬을 형성했는가'** 에 더 크게 좌우된다

## 🔍 분석 과정
1. **분석 마트 구축 (SQL)** — 재주문율·장바구니 크기·구매 주기 등 파생 지표 정의 및 뷰 생성
2. **리텐션 곡선 분석** — 주문 회차별 유저 잔존율 계산, 이탈 집중 구간 파악
3. **고리텐션 vs 일반 유저 비교** — 총 주문 수 상위 20%(23회 이상)를 고리텐션 그룹으로 정의하고, 장바구니 크기·구매 주기·재주문율을 비교해 리텐션 결정 요인 검증
4. **구매 리듬 형성 시점 분석** — 7일 주기가 장기 사용의 결과인지 초기 신호인지 검증

## 💡 핵심 인사이트

**1. 이탈 병목은 2→3 주문 구간이다**
- 전체 유저의 **94%** 가 2번째 주문까지 도달하지만, 주요 이탈은 2→3 구간에 집중
- 이 구간을 넘긴 유저의 **75%** 는 이후에도 구매를 지속

**2. 리텐션을 가르는 건 객단가가 아닌 '구매 리듬'이다 — 반전 인사이트**

| 구분 | 고리텐션 (Top 20%) | 일반 |
|------|------|------|
| 평균 주문 수 | 42회 | 8회 |
| 구매 주기 | **~7일** | ~12일 |
| 재주문율 | **68%** | 39% |
| 장바구니 크기 | **유의미한 차이 없음** | - |

> "얼마나 많이 샀느냐"가 아니라 "얼마나 일정하게 돌아왔느냐"가 리텐션을 결정한다

**3. 7일 리듬은 장기 사용의 결과가 아닌, 초기부터 형성된 신호다**
- 고리텐션 그룹의 7일 구매 주기는 **3번째 주문 시점에 이미 형성**
- 즉, 7일 리듬은 오래 쓴 사람의 특성이 아니라, 장기 고객이 될 가능성을 보여주는 **초기 행동 신호**

**4. 상품(Produce·Dairy·Pantry)은 리텐션의 원인이 아닌 신호다**
- 같은 상품을 구매하더라도, 고리텐션 유저는 '주간 루틴'으로 / 일반 유저는 '단발 이벤트'로 소비
- 상품 카테고리 자체가 리텐션을 만드는 것이 아님

## 📋 전략 제언
전략의 초점은 구매 유도가 아닌, **다시 돌아오게 만드는 재방문 구조 설계**

| 액션 | 내용 |
|------|------|
| 6~7일 리마인드 | 2번째 주문 후 6~7일차에 자주 구매한 상품 중심 푸시 알림 |
| 장바구니 자동화 | "늘 사시던 상품, 장바구니에 담아두었습니다" UX 강화 |
| 초기 경험 설계 | 재구매 주기가 명확한 카테고리를 첫 경험으로 유도 → 루틴 정착 가속 |
