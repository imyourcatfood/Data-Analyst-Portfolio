# Instacart 고객 리텐션 분석

## 프로젝트 개요
Instacart 주문 데이터를 활용하여 주문 단계별 리텐션과 재구매 패턴을 분석한 프로젝트입니다. 반복 구매 행동이 형성되는 시점을 파악하고, 리텐션 개선에 도움이 되는 인사이트를 도출하는 데 목적이 있습니다.

## 데이터 소스
- Instacart 주문 데이터셋
- 주문, 상품, 사용자 행동 관련 테이블

## 분석 질문
- 고객은 몇 번째 주문부터 반복 구매 성향을 보이기 시작하는가?
- 주문 단계별 리텐션은 어떻게 변화하는가?
- 재구매 비율이 높은 상품군 또는 패턴은 무엇인가?

## 사용 도구
- Python
- SQL
- Jupyter Notebook

## 핵심 인사이트
- 주문 회차에 따라 고객 유지율이 어떻게 달라지는지 구조적으로 확인
- 재구매 행동이 강화되는 구간을 파악해 리텐션 개선 포인트 제시
- 반복 구매와 연관된 상품 소비 패턴을 해석 가능한 형태로 정리
