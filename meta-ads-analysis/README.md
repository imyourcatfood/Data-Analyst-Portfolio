## 📌 프로젝트 개요
원석팔찌·커플세트를 주력으로 하는 주얼리 이커머스 A사의 Meta 광고 데이터를 분석한 프로젝트이다.
A사는 전체 매출의 약 55%가 Meta 광고에서 발생하는 **광고 의존형 구조**로, 구매 목표 광고의 **74%가 Meta 알고리즘 미최적화 상태**에서 운영되고 있었다.

분석 목적은 크게 3가지이다:
1. Meta 광고 알고리즘 최적화와 관련된 **비효율 구간 발견**
2. 실제 성과를 만드는 **메시지·소재 전략 도출**
3. 재현 가능한 **광고 운영 기준 수립**

## 🔍 분석 과정

### PART 1 — 광고 성과 구조 분석 (Performance)
1. **Meta 광고 최적화 기준 정의** — 7일 이후 50회 전환 달성 시 알고리즘 학습 완료로 분류
2. **최적화 vs 미최적화 비교** — 구매 광고 74%가 미최적화, 트래픽 광고는 75%가 최적화 완료
3. **미최적화 광고비 비중 분석** — 전체 예산의 약 22%가 학습 미달 상태에서 소진
4. **잠재 광고 식별** — 7일 이전 조기 종료 광고 중, 최적화 광고 초기 성과 Q1 기준 비교로 잠재력 판별
5. **비효율 광고 식별** — 학습 완료 광고에서도 광고비↑ 성과↓인 4사분면 광고 식별

### PART 2 — 광고 콘텐츠 분석 (Content)
1. **메시지 유형 분류** — 의미형(68%) / 제품 강조형(17%) / 이벤트형(15%)
2. **메시지 유형별 성과 비교** — 제품 강조형이 전 지표 1위, 의미형은 비중 대비 효율 낮음
3. **의미형 심층 분석** — 관계 기반(커플/가족) > 자기 중심(SELF), 행복(HAPPINESS) > 행운(LUCK)
4. **메시지 조합 사분면 분석** — 행복+관계 조합이 CTR·ROAS 모두 상위 영역에 집중
5. **ROAS 시뮬레이션** — LUCK→HAPPINESS 메시지 전환 시 ROAS 최대 +32% 개선 추정
6. **소재 유형 확장 분석** — 제품효익 / 관계상황 / 콘텐츠표현 / 프로모션 4가지로 재분류
7. **제품별 최적 소재** — 원석팔찌=제품 가치·효익 중심 / 커플세트=감성 스토리·콘텐츠 중심

### PART 3 — 광고 운영 전략 도출 (Strategy)
- 3단계 실행 로드맵: 즉시 실행 → 1개월 내 → 3개월 내

## 💡 핵심 인사이트

**1. 잠재 광고 발굴 기준 수립**
- 7일 이전 조기 종료된 광고 중, 초기 3-6일 구매 5-8건 또는 ROAS 1.5+ 달성 시 **최소 7일 유지 권장**
- 학습 기간을 확보하여 알고리즘 최적화 기회를 보장하는 운영 기준 도출

**2. 비효율 광고 식별 및 관리 기준**
- 최적화 완료 광고 중에서도 ROAS 2.0↓ & 누적 광고비 상위인 **4사분면 광고**는 소재 교체 또는 예산 축소 대상
- 비효율 예산을 성과 상위(2사분면) 광고로 재배분하는 구조 제안

**3. 메시지 구조가 성과를 결정한다**
- 의미형 메시지(전체 80%) 중 **행복+관계 조합**(ROAS 4.59)이 행운 메시지(ROAS 2.23) 대비 압도적 효율
- 행운→행복 메시지 구조 전환만으로 **추가 광고비 없이 ROAS 최대 +32% 개선** 가능성 시뮬레이션

**4. 제품별 최적 소재 전략이 다르다**
- 원석팔찌: 제품 정보·효익 중심 메시지가 전환 우수
- 커플세트: 감성 스토리·관계 중심 콘텐츠가 ROAS 압도적
- 핵심 타겟 35~54세 여성(매출 61.6%)에 대해 **소재 역할 분리(유입/핵심/전환)** 전략 도출

**5. 3단계 실행 로드맵 제시**
- 즉시: 조기 종료 방지 기준 + 비효율 광고 관리 기준 적용
- 1개월 내: 예산 재배분 + HAPPINESS A/B 테스트 + 제품 강조형 소재 비중 확대(13%→30%)
- 3개월 내: 성과 광고 스케일업 + 운영 기준 재설정 + 주간 점검 프로세스 내재화

> **시사점:** 광고 성과는 운영 구조 + 콘텐츠 구조 + 맥락 전략이 맞을 때 만들어진다
