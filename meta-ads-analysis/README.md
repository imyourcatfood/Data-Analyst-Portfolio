# 주얼리 이커머스 메타 광고비 절감 방법 분석

## 프로젝트 개요
주얼리 이커머스 광고 데이터를 바탕으로 메타 광고 성과를 점검하고, 비용 절감 가능 지점을 찾기 위한 프로젝트입니다. 현재 진행 중이며 이후 분석 결과를 단계적으로 정리할 예정입니다.

## 데이터 소스
- 메타 광고 성과 데이터
- 캠페인, 광고 세트, 소재 단위 운영 데이터

## 분석 질문
- 광고비 효율이 낮은 구간은 어디에서 발생하는가?
- 캠페인 구조와 예산 배분은 어떻게 개선할 수 있는가?
- 비용 절감과 성과 유지 사이에서 어떤 우선순위를 둘 수 있는가?

## 사용 도구
- Spreadsheet / CSV 정리
- SQL 또는 Python
- 성과 분석용 대시보드 도구

## 핵심 인사이트
- 진행 중

## 산출물 링크
- 추후 업데이트 예정

## 회고
- 진행 중인 프로젝트로, 분석 완료 후 문제 정의와 인사이트를 구체화할 예정입니다.
