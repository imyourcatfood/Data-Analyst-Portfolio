## 📊 Projects

🟥 Instacart 고객 리텐션 분석 ([Instacart Customer Retention Analysis](./instacart-analysis))

Instacart 주문 데이터를 활용하여 주문 단계별 리텐션과 재구매 패턴을 분석한 프로젝트입니다. Python과 SQL을 사용해 주문 번호 기준 리텐션 구조와 재구매 비율을 분석하고, 고객의 반복 구매 행동이 형성되는 구간을 파악하여 리텐션 개선을 위한 제안점을 도출했습니다.

This project analyzes customer retention and reorder patterns using the Instacart order dataset. 🟥 중국 해외직구 패션 카테고리 성장 요인 분석 ([Chinese Cross-border E-commerce Analysis](./china-ecommerce-analysis))

중국 이커머스 플랫폼의 해외직구 패션 카테고리 성장 배경을 분석한 프로젝트입니다. Python을 활용한 웹 스크래핑과 데이터 수집을 통해 상품 가격과 시장 데이터를 분석하고, 중국 패션 상품이 해외 시장에서 빠르게 확산된 요인을 가격 경쟁력과 시장 구조 측면에서 살펴보았습니다.

<details>
<summary>English</summary>

This project explores the factors behind the growth of the fashion category in Chinese cross-border e-commerce platforms. 🟥 서울 아파트 가격 회복 분석 대시보드 ([Seoul Apartment Price Recovery Analysis](./seoul-apartment-analysis))

금리 변화에 따른 아파트 가격 흐름을 분석하기 위해 회복지수와 회복률 지표를 설계하고, 매물의 가격 회복 가능성을 탐색할 수 있는 Tableau 대시보드를 구축했습니다. 전고점·전저점·현재가를 기반으로 가격 회복 정도를 정량화하고, 회복지수와 회복률을 함께 비교하여 잠재적인 가격 회복 매물을 탐색할 수 있도록 시각화했습니다.

<details>
<summary>English</summary>

This project analyzes apartment price trends in relation to interest rate changes. I designed recovery indicators—Recovery Index and Recovery Rate—and built a Tableau dashboard to explore listings with strong price recovery potential. Using the historical peak price, trough price, and current price, the dashboard quantifies the degree of price recovery and visualizes both indicators to help identify apartments with potential for price rebound.

</details>
