# 중국 해외직구 패션 카테고리 성장 요인 분석

## 프로젝트 개요
중국 이커머스 플랫폼의 해외직구 패션 카테고리 성장 배경을 분석한 프로젝트입니다. 상품 가격과 시장 데이터를 바탕으로, 해외 시장에서의 확산 요인을 가격 경쟁력과 시장 구조 측면에서 살펴보았습니다.

## 데이터 소스
- 웹 스크래핑으로 수집한 상품 데이터
- 시장 동향 및 가격 비교 데이터

## 분석 질문
- 중국 패션 상품은 어떤 가격 경쟁력을 가지고 있는가?
- 해외직구 시장에서 성장한 배경에는 어떤 구조적 요인이 있는가?
- 가격과 시장 확산 사이에는 어떤 관계가 있는가?

## 사용 도구
- Python
- Web Scraping
- Spreadsheet / CSV 정리

## 핵심 인사이트
- 가격 경쟁력과 시장 구조가 동시에 작동하는 성장 배경을 정리
- 상품 단위 데이터와 시장 정보를 연결해 해석 범위를 확장
- 가격 기반 확산 전략을 데이터 관점에서 설명할 수 있도록 구조화

## 산출물 링크
- 추후 업데이트 예정

## 회고
- 수집 데이터의 품질 관리와 해석 근거를 더 명확히 문서화해 재현성을 높일 계획입니다.
