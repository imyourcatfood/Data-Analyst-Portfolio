# 서울 아파트 가격 회복 분석 대시보드

## 프로젝트 개요
금리 변화에 따른 서울 아파트 가격 흐름을 분석하기 위해 회복지수와 회복률 지표를 설계하고, 가격 회복 가능성을 탐색할 수 있는 Tableau 대시보드를 구축한 프로젝트입니다.

## 데이터 소스
- 서울 아파트 매물 및 가격 데이터
- 시계열 가격 추이 데이터

## 분석 질문
- 현재 매물은 과거 전고점과 전저점 대비 어느 수준까지 회복되었는가?
- 회복지수와 회복률을 함께 보면 어떤 매물이 눈에 띄는가?
- 금리 변화와 가격 회복 흐름을 어떻게 함께 해석할 수 있는가?

## 사용 도구
- Tableau
- Spreadsheet / CSV 정리
- 데이터 전처리 도구

## 핵심 인사이트
- 가격 회복 정도를 단순 상승률이 아닌 보조 지표로 해석 가능하게 정리
- 회복 가능성이 높은 매물을 탐색할 수 있는 시각적 기준 제시
- 시계열 흐름과 현재가를 함께 고려하는 관점으로 대시보드 구성
