# 🏢 서울 아파트 저평가 매물 탐색 대시보드
**금리 사이클 기반 독자 지표로 저평가 가능성 매물을 인터랙티브하게 탐색**

📅 2026.01.28 ~ 2026.02.13 | 팀 프로젝트 (3인)

---

## 📌 프로젝트 개요
단순 시세 비교를 넘어, **금리 사이클을 기준으로 전고점·전저점을 정의**하고
가격 회복 가능성을 정량적으로 평가할 수 있는 독자 지표(회복률·회복지수)를 설계.
이를 바탕으로 저평가 매물을 인터랙티브하게 탐색하는 Tableau 대시보드를 구축했다.

## 📐 지표 설계
기존 단순 시세 비교로는 "지금이 저평가인지 아닌지"를 판단하기 어렵다는 문제에서 출발.
금리 사이클(2022년 급격한 기준금리 인상 → 2023년 이후 안정화)을 전고점·전저점 정의 기준으로 삼아
회복 여력과 반등 속도를 동시에 담을 수 있는 두 가지 독자 지표를 설계했다.

| 지표 | 계산식 | 해석 |
|------|--------|------|
| **회복률** | 현재 평단가 ÷ 전고점 평단가 | 낮을수록 고점 대비 미회복 상태 |
| **회복지수** | (현재가 − 전저점) ÷ (전고점 − 현재가) | 높을수록 바닥 대비 반등폭이 큼 |

→ **회복률이 낮고 회복지수가 높은 매물** = 저평가 가능성이 높은 매물

---

## 🎛️ 대시보드 구성
사용자가 "내가 원하는 조건의 저평가 매물"을 직접 찾을 수 있도록 단계적 필터 구조로 설계했다:

- **구 단위 선택** — 서울 지도에서 관심 지역 클릭 → 해당 구 매물 목록 자동 표시
- **평형·가격 범위 필터** — 구매 가능 예산과 원하는 평형에 맞춰 좁히기
- **지도 위치 확인** — 필터링된 매물의 실제 위치를 지도에서 실시간 확인

---

## ⚠️ 한계점
- 가격 요인만 고려 — 직주근접, 재건축 가능성, 연식, 단지 규모 미반영
- 평단가 기준이므로 소형 평수가 상대적으로 과대평가될 가능성
- 금리 외 거시경제 변수(공급 정책, 계절성) 미고려
